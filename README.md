# 🚀 Test Data Management System (TDMS)

> A full-stack platform to manage, version, and execute test data for automated and manual testing workflows.

---

## 🧠 Why this exists

Modern testing suffers from:

* Scattered test data
* No version control
* Poor reproducibility
* Manual + automated workflows disconnected

**TDMS solves this by centralizing test data, execution, and reporting into one system.**

---

## ⚡ Core Features

* 🧪 Automated + Manual Test Case Execution
* 📦 Dataset Management & Versioning
* ⚡ Real-time Execution (WebSockets)
* 📊 Execution Logs + Reports + Screenshots
* 🌐 Demo Web App for live testing
* 🔌 REST API-driven architecture

---

## 🏗️ System Architecture

```
Frontend (React + Vite)
        ↓
Backend (FastAPI)
        ↓
Database (SQLAlchemy ORM)
        ↓
Execution Engine (Automation + WebSockets)
        ↓
Demo App (Flask target system)
```
